# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

This is a local-only application designed for personal use. 

**WARNING**: This application includes **NO AUTHENTICATION** mechanisms. 
- Do not deploy this application on a public network.
- Do not expose port 8000 to the internet.
- Do not use this application to store sensitive PII (Personally Identifiable Information) or HIPAA-regulated data.

If you discover a security vulnerability, please open an issue in the repository.
